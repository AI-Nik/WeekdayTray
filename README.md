### WeekdayTray
### День недели на панели задач
Простое отображение дня недели в трее. / Simple display of the day of the week in the tray.

### Ru
**Проблема:** Если включены маленькие значки на панели задач, то стандартные часы Windows 10/11 не показывают день недели в формате ddd / Пн, Вт и т.д.

**Решение:** эта программа добавляет в системный трей (рядом с часами) иконку с текущим днём недели: Пн, Вт, Ср, Чт, Пт, Сб, Вс.

![Скриншот](screenshot.png)  (если есть картинка)

### Особенности
- Цвет текста — белый на тёмной панели.
- При наведении мыши всплывает полное название дня и дата.
- Правый клик — выход из программы. Настройки отсутствуют.
- Один файл в автозагрузке.

### Как установить
1. Скачайте `WeekTray.exe` из раздела [Releases](https://github.com/ваш_логин/WeekTray/releases).
2. Запустите один раз. Иконка появится в трее.
3. Чтобы программа запускалась автоматически с Windows:
   - Нажмите `Win+R`, введите `shell:startup`
   - Скопируйте `exe` в открывшуюся папку, либо создайте ярлык этого файла в папке автозагрузки.

### Как изменить иконки
- Измените иконки в папке Icons.
- Замените содержимое Resources в проекте.
- Пересоберите.

### Системные требования
- Windows 10 / 11 (32 или 64 бит)
- .NET Framework 4.7.2 или выше (обычно уже установлен)
